Class Model

Description
Description | Descendents | Methods (details)

Abstract class for model classes. There are two abstract function

that have to be implemented for custom handling. @see Model::process($value) function responsible for process the response (i.e. store data) from client and @see Model::access($value) function responsible for get values from storage
